# 隐私政策

**最后更新日期: 2026-04-22**

欢迎使用特特歌词（以下简称"本应用"）。我们重视您的隐私，本隐私政策旨在向您说明本应用如何收集、使用、存储和保护您的信息。

### 1. 信息收集与使用

#### 1.1 音乐播放数据

本应用会访问与当前播放内容相关的信息，包括：

- 当前播放的歌曲标题、艺术家、专辑名称
- 播放进度和播放状态
- 专辑封面图片
- 音乐服务提供的歌曲标识符

这些数据主要用于在您的个人热点网络中向已连接设备展示当前播放信息和同步歌词。为获取歌词，本应用可能会向第三方歌词服务提供商发送匹配歌词所需的歌曲信息。除实现上述功能所必需的处理外，本应用不会建立自己的服务器来收集、存储或分析您的播放数据。有限的使用分析和崩溃诊断会通过 Firebase 单独处理，详见第 1.8 节。

#### 1.2 媒体库访问

本应用需要访问您的 Apple Music 媒体库以获取当前播放歌曲信息。此权限仅用于读取播放状态，不会修改您的媒体库内容。

#### 1.3 Spotify 集成

若您选择连接 Spotify，本应用将：

- 通过 Spotify 官方授权流程连接您的 Spotify 账户
- 在设备本地保存连接 Spotify 所需的授权信息
- 仅访问当前播放状态和基本播放控制功能

Spotify 授权信息仅用于连接 Spotify 服务，不会分享给无关第三方。

#### 1.4 歌词获取

为提供歌词显示功能，本应用会向第三方歌词服务发送必要的歌曲信息以检索匹配的歌词。

发送的数据通常限于匹配歌词所需的信息，例如歌曲名称、艺术家名称、专辑名称或播放进度。本应用不会主动向歌词服务提供商发送您的姓名、邮箱、电话号码等个人身份信息。

#### 1.5 本地网络服务

本应用会在您的设备上提供本地访问页面，通过您的个人热点网络向已连接设备展示音乐播放信息和歌词。该本地访问功能：

- 仅在您的个人热点局域网内可访问
- 仅用于向您允许连接热点的设备展示歌词页面
- 不作为公开互联网服务提供

#### 1.6 设备偏好设置

本应用会在设备本地存储您的界面偏好设置（如主题、字体大小、歌词显示选项等）。这些数据不会离开您的设备。

#### 1.7 试用期管理

本应用会在设备本地保存试用期使用记录（如已使用时长和重置日期），以确保试用期的公平使用。此类数据：

- 仅存储在您的设备上
- 不包含任何个人身份信息
- 不会传输至任何外部服务器
- 即使卸载应用也可能保留在设备上

#### 1.8 Firebase 使用分析与崩溃报告

本应用使用 Google 提供的 Firebase Analytics 和 Firebase Crashlytics，用于了解应用使用情况、诊断错误和崩溃、衡量稳定性，并改进本应用。

使用分析可能包括：

- 应用使用时长、音乐来源类别、连接步骤、连接成功或中断状态、已连接浏览器/设备类别、歌词可用性状态，以及歌词预览设置变更
- 错误诊断信息，例如经过清理的错误上下文、错误域、错误代码、提供商名称、操作名称、状态码和原因
- Firebase 生成的技术元数据，例如应用版本、系统版本、设备型号、语言/地区、时间戳和 Firebase 安装标识符

崩溃报告可能包括崩溃堆栈、崩溃时间、应用和系统版本、设备型号，以及 Firebase 或 Crashlytics 安装标识符。

本应用不会主动在 Firebase Analytics 事件或由应用提供的崩溃诊断字段中收集或记录歌曲标题、艺术家名称、歌词内容、Spotify 授权令牌、Apple Music 令牌、IP 地址或原始浏览器标识。分析事件参数会在发送前进行清理。

本应用提供「共享使用分析」设置。在需要征求同意的地区，本应用会在启用使用分析前请求您的许可；在其他地区，使用分析可能默认启用，您可以在 iOS「设置」>「特特歌词」>「共享使用分析」中关闭。关闭后，本应用会停用 Firebase Analytics 收集，并重置设备上的 Firebase Analytics 数据。

Firebase 数据会传输至 Google 并由 Google 按其 Firebase 和 Google Analytics 条款及隐私实践处理，包括：

- [Firebase 隐私与安全](https://firebase.google.com/support/privacy)
- [Google 隐私权政策](https://policies.google.com/privacy)
- [Google Analytics 服务条款](https://www.google.com/analytics/terms/)
- [Firebase Crashlytics 条款](https://firebase.google.com/terms/crashlytics/)

### 2. VPN 功能说明

**重要声明：** 本应用使用本地 VPN 功能帮助个人热点中的已连接设备访问歌词显示页面，其目的**不是**提供互联网隐私保护、流量加密或匿名浏览服务。

本应用的 VPN 功能：

- **仅用于**在您的个人热点网络中创建本地访问通道，使连接热点的设备能够访问歌词显示页面
- **不会**路由、拦截、监控或修改您的互联网流量
- **不会**将您的流量发送至任何外部服务器或代理
- **不会**改变您的 IP 地址或地理位置
- **不会**收集或分析您的网络活动

VPN 相关处理仅用于支持本地歌词页面访问，不用于代理、分析或转发您的互联网流量。

### 3. 数据存储

| 数据类型 | 存储位置 | 保留期限 |
|---------|---------|---------|
| 播放状态 | 设备内存 | 仅运行时，关闭即清除 |
| 界面设置 | 设备本地存储 | 直到您卸载应用或手动重置 |
| Spotify 授权信息 | 设备本地存储 | 连接有效期内保留，失效后需要重新授权 |
| 试用期记录 | 设备本地安全存储 | 为管理试用期可能长期保留 |
| 歌词缓存 | 设备内存 | 仅运行时，关闭即清除 |
| Firebase Analytics 事件 | Google Firebase / Google Analytics | 按 Google/Firebase 的保留设置和政策保留 |
| 崩溃诊断信息 | Google Firebase Crashlytics | 按 Firebase Crashlytics 政策保留 |

### 4. 数据共享

本应用**不会**：

- 向任何第三方出售、出租或交换您的个人信息
- 收集或传输设备标识符（IDFA、IDFV 等）用于广告目的
- 将 Firebase 数据用于个性化广告、用户画像或定向推送
- 将歌曲标题、艺术家名称、歌词内容、Spotify 授权令牌、Apple Music 令牌、IP 地址或原始浏览器标识作为 Firebase 的主动分析或应用提供的诊断数据分享

本应用会按第 1.8 节所述向 Google Firebase / Google Analytics 分享有限的使用分析和崩溃诊断信息。本应用也可能按第 1.4 节所述向歌词服务提供商发送歌曲信息。

### 5. 数据安全

本应用采取以下措施保护您的数据：

- 本地访问页面仅面向连接到您个人热点的设备
- 试用期数据使用系统提供的安全存储能力保存
- Firebase Analytics 已配置为停用广告存储、广告用户数据和广告个性化
- 分析事件名称和参数在发送前会被限制和清理
- 不维护任何远程数据库或用户账户系统

### 6. 儿童隐私

本应用不针对 13 岁以下儿童，也不会故意收集儿童的个人信息。

### 7. 隐私政策变更

我们可能会不时更新本隐私政策。更新后的政策将在应用内或应用商店页面公布，继续使用本应用即表示您同意修订后的政策。
